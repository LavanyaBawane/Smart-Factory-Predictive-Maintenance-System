# Smart Factory Predictive Maintenance System

## Overview
This project aims to develop a Smart Factory Predictive Maintenance System using ESP32, TinyML, MQTT, and Generative AI.

The system will monitor machine vibrations, detect anomalies at the edge using TensorFlow Lite, transmit alerts through MQTT, and generate maintenance recommendations using an LLM.

## Objectives
- Real-time vibration monitoring
- Edge AI anomaly detection
- MQTT communication
- Predictive maintenance alerts
- AI-generated maintenance recommendations
